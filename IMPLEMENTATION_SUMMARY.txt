API ENDPOINTS
================================================================================

PUBLIC:
  GET  /health           - Health check (always 200 if running)
  GET  /ready            - Readiness check (503 if not ready)
  GET  /metrics          - Prometheus metrics (text format)

AUTHENTICATED (Basic Auth):
  GET  /api/v1/queue/stats       - Queue statistics
  GET  /api/v1/queue/pending     - List pending messages
  GET  /api/v1/dlq/list          - Dead letter queue
  POST /api/v1/dlq/retry/{id}    - Retry failed message
  GET  /api/v1/message/{id}      - Get message details
